# El Arte de la Limpieza de Datos con R 📊

Como estadístico, entiendo que un modelo es tan bueno como los datos que lo alimentan. Este proyecto demuestra mi flujo de trabajo para transformar bases de datos ruidosas en información lista para el análisis.

### Capacidades Técnicas:
- **Tratamiento de NAs:** Implementación de imputación de datos mediante la mediana para conservar la estructura de la muestra.
- **Tidyverse:** Uso experto de dplyr y readr para la manipulación eficiente de data frames.
- **Rigor Estadístico:** Creación de variables derivadas (como IMC) y normalización de registros para análisis demográficos y financieros.

"Limpiar datos no es borrar errores; es preparar el escenario para que la verdad estadística pueda brillar."
